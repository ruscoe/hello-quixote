A plugin that displays quotes from Don Quixote in the admin footer.

== Description ==

Fight all your WordPress admin tasks on the back of the trusty Rocinante by
installing a plugin that displays quotes from Miguel de Cervantes' Don Quixote
in the admin footer!

== Installation ==

1. Copy the plugin to `/wp-content/plugins/hello-quixote`
2. Activate the Hello Quixote plugin through the 'Plugins' screen in WordPress
